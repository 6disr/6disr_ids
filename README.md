ID System

A lightweight and immersive player ID system designed for roleplay servers that want to reduce screen clutter and discourage metagaming.

Features
Hidden by Default – Player IDs are never permanently displayed.
Keybind Activated – IDs are only visible while holding the configured key.
Vehicle Aware – Players inside vehicles will not have their ID displayed.
Immersive – Keeps interactions natural by preventing identification through vehicle windows.
Optimized – Built to be lightweight with minimal performance impact.
